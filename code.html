How the Algorithm Decodes Truth

Phase 1

Syntax Integrity Check

Before deep NLP runs, the system measures raw data integrity. If a user inputs insufficient data (like short random keystrokes or "abcd"), the AI immediately drops the process and flags it as a false input to prevent mathematical manipulation of the score.

Phase 2

Global Ledger Cross-Sync

The engine extracts the domain from the URL and pings it against a hardcoded, live global ledger. It instantly identifies trusted journalistic broadcasters, isolates known satirical networks, and heavily penalizes flagged phishing domains.

Phase 3

Deep NLP & Structural Typology

We don't just read words; we analyze psychological intent. Our algorithms scan for hyperbole, fear-mongering syntax, and manipulative trigger words. Simultaneously, it measures structural integrity—like excessive punctuation and ALL CAPS screaming.
