**Verify the Truth in Milliseconds.** InfoVerify is a high-performance, frontend web application designed to simulate advanced Natural Language Processing (NLP) and source-verification to detect clickbait, fabricated news, and phishing threats.

🌐 **[Live Demo](https://jrajimman12.github.io/infoverify.app/)**

---

## ✨ Core Features

* **🧠 Simulated NLP Engine:** Uses a custom deterministic hashing algorithm to evaluate linguistic topologies, emotional trigger words, and formatting anomalies (e.g., excessive punctuation, ALL CAPS).
* **🌐 Source Cross-Referencing:** Pings inputted URLs against a hardcoded ledger of trusted journalism broadcasters, satirical sites, and flagged phishing domains.
* **🛡️ Syntax Integrity Validation:** Automatically detects and rejects gibberish or insufficient data inputs to prevent mathematical manipulation of the trust score.
* **📡 Live Threat Interception Feed:** Features a continuously scrolling CSS marquee displaying intercepted threat vectors in real-time.
* **👁️ The "Watcher" Drone:** An interactive, mathematics-driven UI element that actively tracks the user's cursor across the screen during data entry.
* **💎 Glassmorphism UI:** Built with an enterprise-grade dark mode interface featuring dynamic scaling, CSS blur filters, and interactive "Mega-Guardian" easter eggs.

---

## 🛠️ Technical Architecture

This project is entirely client-side, requiring no external backend or databases to function, making it highly portable and fast.

* **Frontend Structure:** HTML5
* **Styling & Animation:** Advanced CSS3 (Flexbox, CSS Grids, Keyframe Animations, Backdrop Filters, Custom Properties)
* **Logic & Interactivity:** Vanilla JavaScript (ES6+)
* **Assets:** [FontAwesome 6.4](https://fontawesome.com/) for iconography, Google Fonts (Inter, Fira Code).
